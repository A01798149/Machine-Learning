# Almacenar los resultados
resultados = {}

# Iterar sobre cada tamao de la lista de sample_sizes
for size in sample_sizes:
    mse_train_list = []
    mse_val_list = []

    for i in range(100):
        # PUNTO 8: Seleccin de un subconjunto aleatorio y entrenamiento del modelo
        X_subset, _, y_subset, _ = train_test_split(X_train, y_train, train_size=size, random_state=seed+i)

        # Definir y entrenar el modelo
        model = SGDRegressor(learning_rate='constant', eta0=1E-4, max_iter=1000000, random_state=seed+i)
        model.fit(X_subset, y_subset)

        # PUNTO 9: Clculo del MSE en el subconjunto de entrenamiento y validacin
        y_train_subset_pred = model.predict(X_subset)
        y_val_pred = model.predict(X_val)

        # Calcular el error cuadrtico medio (MSE)
        mse_train = mean_squared_error(y_subset, y_train_subset_pred) # Calcula el MSE para el subconjunto de entrenamiento comparando los valores reales y_subset con las predicciones y_train_subset_pred
        mse_val = mean_squared_error(y_val, y_val_pred)               # Calcula el MSE para el conjunto de validacin comparando los valores reales y_val con las predicciones y_val_pred

        mse_val_list.append(mse_val)

    # Almacenar los MSE para este tamao de instancia

  • Error Promedio en Entrenamiento: El error cuadrtico medio (MSE) es muy bajo en comparacin con los otros modelos, lo que indica un ajuste casi perfecto a las dos muestras de entrenamiento seleccionadas.   • Error Promedio en Validacin: El MSE en validacin es extremadamente alto, lo que confirma que el modelo no generaliza adecuadamente a datos no vistos. Este comportamiento es caracterstico de un modelo con alto sesgo (bias) debido a la falta de suficientes datos para capturar la verdadera relacin entre las variables. El modelo tiene un alto nivel de varianza, reaccionando de manera excesiva a las pequeas variaciones en los datos de entrenamiento.

En pocas palabras, este muestra un comportamiento de sobreajuste extremo con alta varianza y alto sesgo.

2. Modelo Final (Línea Base - 40 Muestras):¶

  • Error Promedio en Entrenamiento: El MSE es relativamente estable y ms bajo en comparacin con otros tamaos de muestra ms pequeos, lo que indica que el modelo est capturando una tendencia general sin sobreajustarse a los datos.
  • Error Promedio en Validacin: Aunque el MSE es un poco ms alto en comparacin con el entrenamiento, sigue siendo estable y bajo, lo que sugiere que el modelo generaliza bien a datos no vistos. Este modelo tiene un buen balance entre sesgo y varianza, logrando capturar las relaciones subyacentes sin reaccionar demasiado a las variaciones especficas del conjunto de datos de entrenamiento.

De manera resumida podemos decir que, en este caso se logra un buen equilibrio entre sesgo y varianza, capturando la relacin subyacente de manera efectiva.

3. Evolución del Ajuste con el Incremento del Número de Muestras:¶

  • Para muestras pequeas (2-6): El modelo muestra una tendencia de sobreajuste con un bajo MSE en entrenamiento y un MSE extremadamente alto en validacin, lo que indica una alta varianza y un alto sesgo. Los modelos con 4, 5 y 6 muestras, son aquellos que tienen los menores valores de error tanto en entrenamineto como validacin.
  • Para muestras intermedias (11-16): Aqu el MSE en validacin comienza a disminuir y estabilizarse, lo que sugiere que el modelo empieza a capturar mejor la relacin entre las variables, reduciendo el sesgo y mejorando su capacidad de generalizacin.

Entonces, a medida que se incrementa el nmero de muestras, el error en validacin disminuye significativamente hasta que se estabiliza, lo que refleja una mejora continua en la generalizacin del modelo. Esto sugiere que, para este conjunto de datos, un tamao de muestra mayor permite al modelo aprender mejor las relaciones verdaderas entre las variables y reducir tanto el sesgo como la varianza.
